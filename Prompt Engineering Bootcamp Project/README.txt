Global AI Hub tarafından gerçekleştirilen "Aygaz GenAI Bootcamp: Prompt Mühendisliğine Giriş" bootcampı için geliştirilmiş Portfolio / Galeri Web Sitesi projesidir.
* Bu projede, yapay zeka araçlarını kullanarak ürettiğiniz görsel ve metinleri, yine bu araçlar yardımıyla geliştireceğiniz HTML tabanlı bir web sitesinde sergilemeniz beklenmektedir. 
* Code prompts için ChatGPT, image prompts için designer.microsoft.com kullanılmıştır.
